# Example for @prosemirror-adapter/react

A simple example for [`@prosemirror-adapter/react`](../../packages/react/).

## Getting Started

1. Clone the repo.

2. Install dependencies by `pnpm install`.

3. Run the example by `pnpm ex:react`.
