# Example for @prosemirror-adapter/vue

A simple example for [`@prosemirror-adapter/vue`](../../packages/vue/).

## Getting Started

1. Clone the repo.

2. Install dependencies by `pnpm install`.

3. Run the example by `pnpm ex:vue`.
